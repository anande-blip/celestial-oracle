# Celestial Oracle - Backend

Multi-oracle AI system with LiveKit video streaming and real-time voice divination.

## Setup

1. Clone repo
2. Create virtual environment: `python3 -m venv .venv`
3. Activate: `source .venv/bin/activate`
4. Install: `pip install -r requirements.txt`
5. Configure: `cp .env.example .env` (add your API keys)
6. Run: `python3 agent.py dev`
